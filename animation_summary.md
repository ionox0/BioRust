## Animation Type Analysis

All animations contain combinations of:
- **Translation** - Position changes
- **Rotation** - Rotational movement
- **Scale** - Size changes (less common)

## Usage Notes

1. **Black Ox Beetle** has the most comprehensive animation set with 49 animations covering all aspects of gameplay
2. **Bee** has specialized flying animations including hover and take-off/landing
3. **Scorpion** has a basic but complete set for typical gameplay
4. **Wolf Spider** focuses on various movement patterns
5. **Spider Small** only has an attack animation - may need additional animations for complete gameplay
6. **Ladybug** animations appear to be placeholder cube animations and may need replacement

## Technical Details

- All files use GLTF 2.0 format
- Animation data is stored in binary GLB format
- Channel counts vary from 2 to 229 per animation
- Duration information is available for most animations (some show "Unknown" if duration couldn't be determined)
